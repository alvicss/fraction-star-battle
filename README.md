# 🚀 分數星際大作戰

> 國小中年級分數練習卡牌遊戲，純前端、無需安裝、直接開啟即玩。

**線上試玩：** https://alvicss.github.io/fraction-star-battle/

---

## 遊戲介紹

宇宙怪獸「亂亂怪」入侵了數學星球，把分數石碑全部打亂！選擇你的腳色（劍士 / 魔法師 / 弓箭手），用分數卡牌對抗敵人，拯救數學星球！

### 5 個關卡

| 關卡 | 名稱 | 練習技能 |
|---|---|---|
| 1 | 🪐 加法星球 | 同分母分數加法 |
| 2 | ☄️ 減法彗星 | 同分母分數減法 |
| 3 | 🌌 乘法銀河 | 分數 × 整數 |
| 4 | 🕳️ 比大小黑洞 | 分數大小比較 |
| 5 | ⭐ 雙星挑戰 | 多步驟策略選牌 |

### 遊戲規則

- 每場最多 10 局，玩家和怪獸各有 5 點血量
- 答對 → 怪獸扣血；答錯 → 玩家扣血
- 結算依血量差給予 0～3 顆星，歷史最高星數會記錄下來

## 技術規格

- **純前端**：單一 `index.html` 檔案，包含所有 CSS + HTML + JavaScript
- **無框架**：不依賴任何前端框架或套件
- **無後端**：不需伺服器，直接用瀏覽器開啟
- **RWD**：支援手機、平板、桌機

## 檔案結構

```
index.html          # 主遊戲（所有程式碼在同一檔案）
SPEC.md             # 功能規格文件
audio/
  intro-bgm.mp3     # 開場背景音樂
  battle-bgm.mp3    # 對戰背景音樂
  stage-clear.mp3   # 過關音效
  stage-fail.mp3    # 失敗音效
  card-play.mp3     # 出牌音效
  card-play-alt.mp3 # 備用音效
picture/
  monster.png       # 宇宙怪物
  swordsman.png     # 劍士
  mage.png          # 魔法師
  archer.png        # 弓箭手
```

## 本機執行

直接用瀏覽器開啟 `index.html` 即可，無需任何建置步驟。
